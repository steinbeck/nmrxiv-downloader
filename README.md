# nmrxiv-downloader

CLI tool for searching and downloading NMR datasets from nmrxiv.org.

## Installation

```bash
pip install -e .
```

## Usage

```bash
nmrxiv --help
nmrxiv search --help
nmrxiv show --help
nmrxiv download --help
```

## Features

- Search NMR datasets by experiment type, format, and molecular properties
- Browse dataset metadata in JSON or human-readable format
- Download datasets to local disk with organized directory structure
